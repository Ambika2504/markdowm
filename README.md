# markdowm
Markdown Notes App is a full-stack web application that allows users to create, edit, delete, and manage notes efficiently.It is built using React for the frontend, Node.js &amp; Express for the backend, and SQLite as the database. The application also includes a Dark Mode feature for a better user experience.
##  Features

-  Create Notes
-  Edit Notes
-  Delete Notes
-  Live Markdown Preview
-  Dark Mode Toggle

##  Tech Stack

### Frontend
- React
- React Markdown

### Backend
- Node.js
- Express
- SQLite

  ## Steps to run this
  ### backend
      - mkdir markdown-notes-app
      - cd markdown-notes-app
      - mkdir backend
      - cd backend
      - npm init -y
      - npm install express cors body-parser sqlite3
      - Paste the backend code server.js
      - node server.js
      - we can see Server running on http://localhost:5000
  ### frontend
      - npx create-react-app frontend
      - cd frontend
      - npm install axios react-markdown
      - Open src/App.js and src/App.css and remove total code and paste the code
      - npm start
